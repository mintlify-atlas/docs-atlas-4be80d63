# Extrator de Tarefas Auvo

A Flask-based web application designed to streamline the analysis of task reports exported from Auvo. Upload CSV files, filter tasks by customizable keywords, and export professional reports in multiple formats.

## How It Works

<Steps>
  <Step title="Upload your CSV file">
    Export a task report from Auvo and upload the CSV file through the web interface.
  </Step>
  <Step title="Automatic filtering">
    The application scans the "Relato" column for your configured keywords and extracts matching tasks.
  </Step>
  <Step title="Review and export">
    View filtered results with statistics, search through tasks, and export to Excel or PDF.
  </Step>
</Steps>

## What You'll Find

Analyze task reports containing keywords that indicate work requiring action, such as equipment failures, part replacements, or installation requests. The tool helps you quickly identify and report on tasks that need follow-up.

<Note>
  The application supports CSV, XLS, and XLSX file formats exported from Auvo.
</Note>
